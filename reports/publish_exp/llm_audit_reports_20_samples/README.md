# LLM Audit Reports for 20-Sample Unauthorized Token Transfer Experiment

This directory contains the Chinese audit reports generated by the proposed off-chain LLM-assisted smart contract
audit system.

## Experiment Scope

- Total samples: 20
- Vulnerable samples: 15
- Safe samples: 5
- Target vulnerability: unauthorized token transfer / authorization-bypass asset transfer
- Model: qwen3-coder:30b via local Ollama
- Workflow: Slither context + LLM audit report generation + Foundry PoC/patch validation

## Important Note

These reports are raw LLM-generated audit outputs from the automated stage. They should not be treated as final
trusted audit conclusions. The paper uses these reports together with compilation, Foundry tests, Slither re-scan,
structural checks, and manual review.

In particular, some reports contain false positives or incorrect vulnerability classifications. These cases are
intentionally preserved to support the paper's conclusion that LLM outputs must be verified before being trusted.
